### 声音配置快速指南（voice_params）

一个地方配置即可：把下列键写进 `voice_params`（字典/JSON），系统会自动生成对应的 SSML。未提供的键将使用默认值。

#### 支持的键

- **contour**（音高轮廓，字符串）
  - 作用：按文本时长百分比指定音高变化轨迹，更适合整句或长语段。
  - 格式举例：`"(0%,+20Hz) (10%,-2st) (40%,+10Hz)"`
  - 建议：不要用于很短的词/短语。

- **pitch**（基准音高）
  - 绝对值：`"600Hz"`
  - 相对值：
    - 以 Hz 或 st 半音为单位：`"+80Hz"`, `"-2st"`
    - 以百分比：`"50%"`, `"-50%"`
  - 枚举：`"x-low" | "low" | "medium" | "high" | "x-high"`
  - 建议范围：原音高的 0.5～1.5 倍（大幅度会失真或不自然）。

- **range**（音高变化范围）
  - 表达方式与 pitch 相同（绝对/相对/百分比/枚举）。

- **rate**（语速）
  - 相对倍数：`"0.5"`（减半）、`"1"`（默认）、`"2"`（加倍）
  - 百分比：`"50%"`, `"-30%"`
  - 枚举：`"x-slow" | "slow" | "medium" | "fast" | "x-fast"`
  - 建议范围：0.5～2（±100% 以内更自然）。

- **volume**（音量）
  - 绝对值（0～1 或 0～100）：`"0.6"` 或 `"60"`（越大越响）
  - 相对量：`"+10"`, `"-5.5"`
  - 百分比：`"50%"`, `"+3%"`
  - 枚举：`"silent" | "x-soft" | "soft" | "medium" | "loud" | "x-loud"`
  - 建议：对话类推荐 `medium` 附近；过大易失真。

#### 配置示例

- JSON/字典（推荐）

```json
{
  "pitch": "+2st",
  "range": "high",
  "rate": "1.2",
  "volume": "medium",
  "contour": "(0%,+20Hz) (50%,-10Hz) (100%,+5Hz)"
}
```

- 只调语速与音量

```json
{
  "rate": "fast",
  "volume": "0.7"
}
```

- 温柔低沉风格

```json
{
  "pitch": "low",
  "range": "x-low",
  "rate": "0.85",
  "volume": "soft"
}
```

#### 使用说明与注意事项

- 不同键可组合使用，系统会自动合成为 `<prosody ...>`；当未设置任何 prosody 参数时，会仅包一层 `<lang xml:lang="...">` 以保证语言正确。
- `contour` 主要用于整句节奏/旋律感的设计，不建议用于很短的文本。
- `pitch/range/rate/volume` 的极端值可能造成不自然或设备端限制，建议渐进调试。
- 你还可以同时指定情绪（emotion），系统会在外层加上 `<mstts:express-as>` 后再应用 prosody。
